Project 7:
Web Design

Project Description

The aim of the project for my third year content management systems project as to design and document a portfolio site detailing my coursework throughout my undergraduate studies. That site is the current site you are currently browsing right now!

My Approach

I created both paper prototypes and Figma prototypes to create provisional designs for my site, as well as conducting an extensive literature review on the history of content management systems, portfolio design practices, heuristic evaluations, competitor analysis, and, most importantly, how to design sites to be accessible. I found that none of the no-code content managemet system available to me were able to achieve what I wished to do, so I coded the site myself in visual studio code using HTML, CSS, and JavaScript. The GitHub repository showing the source code is available below.
